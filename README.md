# GameDevRoadmap

GameDevRoadmap é um site que organiza uma sequência progressiva de jogos simples para estudo e prática de desenvolvimento de jogos.

O objetivo é oferecer um caminho realista para desenvolvedores iniciantes que desejam aprender os fundamentos do game development sem a ilusão de criar, logo no início, projetos gigantescos e inviáveis.

## Para quem é este projeto

- Desenvolvedores iniciantes
- Pessoas estudando programação com interesse em jogos
- Autodidatas que querem aprender fundamentos, não apenas ferramentas
- Quem aceita começar pequeno para evoluir de forma sólida

## Como funciona

O site apresenta uma lista de jogos organizados por complexidade crescente.  
A proposta é simples:

1. Comece pelo primeiro jogo da lista  
2. Implemente com ferramentas simples  
3. Refaça o mesmo jogo em outras linguagens ou tecnologias  
4. Evolua gradualmente a complexidade

## Sobre ferramentas e engines

O projeto **não é dependente de nenhuma engine**.

Recomendamos fortemente que os primeiros jogos sejam desenvolvidos sem engines famosas como Unity ou Godot.  
A ideia é começar com tecnologias simples (como HTML, Canvas, JavaScript ou linguagens básicas), focando em:

- Lógica
- Estruturas de dados
- Loop de jogo
- Colisão
- Estado
- Entrada do jogador

Somente depois, faz sentido migrar para engines mais complexas.

## Filosofia do projeto

GameDevRoadmap não é sobre criar o jogo dos sonhos imediatamente.  
É sobre construir base técnica, entendimento profundo e autonomia.

O desenvolvimento de jogos é vasto, complexo e bonito demais para ser reduzido a tutoriais rápidos ou limites de mercado.

Comece pequeno. Refaça. Mude de linguagem. Aprenda os fundamentos.
